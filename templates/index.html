Welcome to the Inventory Admin Web App

Glad that you stopped by to check out my inventory administrative application. This is a project that I undertook to introduce innovative ideas into the workplace. What I have created is just a prototype, so please be patient as you go through the web application.

For the past three years, I have been drawing concept art all over any blank canvas that is easily accessible. All in attempt to be able to accurately portray the ideas that I have in my head to make documentation easier for all those involved. This web application is a manifestation of those ideas.

The purpose of this program is to innovate the audit documentation process to:

  • Recording audit documentation to an integrated database
  • Easy retrieval of needed information
  • Data needs automatically generated instead of manually compiled
  • Consistent formatting across audit engagements
  • Intuitive design for ease of use by all end users
  • Data in readable format for validation and AI purposes

Please take a look around. Hopefully you have as much fun learning about this program as I did designing it.

Christiaan Young
